Sistema de Gestión de Personas en Java
Proyecto desarrollado en Java aplicando conceptos de Programación Orientada a Objetos, incluyendo herencia, composición, encapsulamiento, constructores, getters/setters y sobrescritura de toString()

Conceptos aplicados
- Herencia: clases como Cliente, Empleado o Partner extienden desde una clase base Persona
- Composición: clases como Rut y Direccion se usan como atributos dentro de Persona
- Encapsulamiento: uso de atributos privados con métodos get y set para acceder a los datos
- toString(): sobrescritura del método para mostrar la información de los objetos de forma legible

Requisitos cumplidos
- Organización en paquetes model y app
- Clase base general (Persona)
- Clase con composición (Direccion o Rut)
- Clases hijas que heredan de la base (Cliente, Empleado, Partner)
- Uso de constructores, getters, setters y toString()

Ejecución
- Abrir el proyecto en IntelliJ IDEA o en otro IDE compatible con Java.
- Verificar que la estructura de paquetes se mantenga correctamente.
- Ejecutar la clase Main ubicada en src/app/Main.java.
